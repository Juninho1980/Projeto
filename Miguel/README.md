
Manual do Usuário: Sistema de Gerenciamento de Pedidos de Pizzaria

Este sistema foi desenvolvido para auxiliar no gerenciamento eficiente dos pedidos
em uma pizzaria. Ele oferece diversas funcionalidades para facilitar o dia a dia do
estabelecimento, como adicionar novos pedidos, consultar o histórico de pedidos,
calcular o valor total das vendas e muito mais.
